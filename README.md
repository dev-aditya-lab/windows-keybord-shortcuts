## Customizing Shortcuts

Windows 11 allows you to customize certain keyboard shortcuts to better fit your workflow.

### Customizing Taskbar Shortcuts

You can pin your favorite apps to the taskbar and then use the **Win + [Number]** shortcut to open them quickly. To pin an app to the taskbar:
1. Open the app you want to pin.
2. Right-click the app icon on the taskbar.
3. Select "Pin to taskbar."

### Creating Custom Shortcuts

You can create custom keyboard shortcuts for any app or file:
1. Right-click the app or file and select "Properties."
2. In the "Shortcut" tab, click the "Shortcut key" box.
3. Press the combination of keys you want to use for the shortcut (e.g., **Ctrl + Alt + M**).
4. Click "OK" to save.
